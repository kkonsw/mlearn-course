# Machine Learning Course

-------------------

## Предсказывание стоимости домов

Dataset - <https://www.kaggle.com/harlfoxem/housesalesprediction>

Dataset contains house sale prices for King County, which includes Seattle. It includes homes sold between May 2014 and May 2015.

1. Разбор, преобразование и визуализация данных.
2. KNN regressor
3. KNN classifier
4. PCA
5. Полносвязная (неглубокая) нейронная сеть
6. Random forests
7. Gradient boosted trees
